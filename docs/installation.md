# Installation

This guide will help you install and configure Inertia I18n in your project.

## Requirements

- Laravel application with Inertia.js setup
- Node.js and npm/yarn
- Supported frontend framework (React or Vue)

## Backend Installation

Install the Laravel package via Composer:

```bash
composer require inertia-i18n/laravel
```

## Frontend Installation

Install the appropriate frontend package:

### For React
```bash
npm install @inertia-i18n/react
```

### For Vue
```bash
npm install @inertia-i18n/vue
```

## Configuration

Configure your application following the framework-specific guides:

- [React Setup](framework/react.md)
- [Vue Setup](framework/vue.md)
